# N figure solver
## An app to solve N queens problem for any figure
### First time doing smth on github other than labs and first somewhat big project with some original idea in mind.
 The idea is doing n queens problem for almost any figure.I implemented backtracking algorithm to create all boards for figure and size of board and local conflicts algorithm for  finding solution for much larger n
Now it is in progress and i will add more functionality but now it has (poor) GUI and can output a file with "board"(to minimize size of a file it outputs coords of a figure if there is one in those coords)
